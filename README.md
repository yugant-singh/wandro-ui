# Wonder — Explore the World

## Live-Demo --> https://wandro-ui.netlify.app/


**Wonder** is a visually engaging travel-themed website designed to showcase the most stunning wonders across the world — from ancient marvels to natural landscapes.  
It’s built using clean HTML, SCSS, and responsive design principles to provide an immersive experience for travelers and explorers.

---

## Features

- Explore famous and hidden wonders from around the world  
- Interactive hero section with background videos  
- Categorized destinations and travel inspiration  
- Trekker’s Highlights section featuring travel stories  
- Newsletter-style “Get Inspired” section  
- Fully responsive and modern UI  
- Organized footer with quick navigation and social links  

---

## Tech Stack

| Technology | Purpose |

| SCSS / CSS3 | Styling and responsive layout |
